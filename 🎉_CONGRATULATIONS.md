## 🚀 **GET STARTED NOW!**

### Step 1: Install
```bash
npm install
```

### Step 2: Configure
Create `apps/web/.env.local`:
```env
NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
NEXT_PUBLIC_API_URL=http://localhost:8083/api/v1
```

### Step 3: Run
```bash
npm run dev:web
```

### Step 4: Explore
Open: **http://localhost:3000**
